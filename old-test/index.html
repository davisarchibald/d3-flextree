Flextree pages

There are tree different test pages here, all of which use the same set of test cases, defined in the cases subdirectory (see the README file there for descriptions).

  • test-d3-fork - tests my fork of the D3 repo, using the modified d3.layout.tree
  • test-d3-flextree - tests the d3-flextree plugin, which is auto-generated from my fork of D3, so the results should be identical.
  • test-d3 - tests only the applicable test cases against the existing (version 3.5.6) D3, to ensure that the new flextree is backwards-compatible.
